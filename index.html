Prototype Writeup

What have we done so far?

So far, we made a first version of the webpage and connected it to the Steam CSV data. The page reads the dataset, groups the games by their first listed genre, and turns those groups into a bar chart. We added buttons so people can switch between looking at game count, average rating, average playtime, and average price. We also added a genre dropdown and hover details so the visualization feels more interactive instead of just being a static chart.

What will be the most challenging part to design and why?

The hardest part will probably be deciding what story the final visualization should focus on. The Steam data has a lot of possible angles, like genre, price, ratings, owners, and playtime, so it is easy for the page to get too crowded. Genres are also a little messy because each game can have multiple genres and tags, which means the grouping method can change what the chart seems to say. We think the main design challenge will be making the page easy to explore while still giving viewers a clear takeaway right away.
